# uniq
Uniq id generator for node

## Install

```
npm install @vasuvanka/json-validator
```

## Docs

* API Documentation: https://vasuvanka.github.io/json-validator

## Example

```js
const { uniq } = require('@vasuvanka/uniq');
uniq() // RiHO71I3St0u

```
